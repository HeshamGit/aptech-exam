17. Which sorting method is represented in the following pseudocode? (Mark is 2.00 - Difficulty is 4)
START;
SIZE ? N;
DEFINE NUMBER(SIZE);
FOR PASS IN RANGE 1 to N-1
DO
     FOR I IN RANGE 1 to (N-PASS)
     DO
         IF NUMBER(I) > NUMBER(I+1) THEN
          TEMP ? NUMBER(I);
          NUMBER(I) ? NUMBER(I+1);
  NUMBER(I+1) ? TEMP;
        ENDIF;
     ENDFOR;
ENDFOR;
DISPLAY ?Array is sorted?;
STOP;

  • Insertion Sort
  • Binary Sort
  • Shell Sort

  • Bubble Sort
